# 🌍 World Happiness Data Analysis (2015)

## 📌 Overview

This project analyzes the **World Happiness Report 2015 dataset** to explore the factors influencing happiness scores across different countries.

Using Python, the dataset is cleaned and examined to understand how variables such as GDP, health, freedom, and social support impact overall happiness levels.

---

## 🎯 Objectives

* Analyze global happiness scores across countries
* Explore relationships between economic and social factors
* Identify key indicators affecting happiness
* Perform data cleaning and preprocessing
* Visualize insights using graphs

---

## 📊 Key Insights

* Countries with higher GDP and strong social support tend to have higher happiness scores
* Health (life expectancy) shows a strong positive correlation with happiness
* Freedom and trust in government also contribute significantly
* Regional differences highlight variations in living standards and well-being

---

## 🛠️ Tools & Technologies

* Jupyter Notebook

---

## 📂 Dataset Features

* Country and region
* Happiness score and rank
* GDP per capita
* Health (life expectancy)
* Freedom
* Trust (government corruption)
* Generosity

## 📸 Visualizations

* Bar charts comparing countries
* Correlation heatmaps
